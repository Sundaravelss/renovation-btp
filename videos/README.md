# Hero Background Video

Place `hero-bg.mp4` in this directory.

## Recommended specs
- Resolution: 1920×1080 or 1280×720
- Duration: 10-30 seconds looping
- Codec: H.264, no audio
- File size: < 8MB for fast loading
- Autoplay friendly: muted, short loop

## Sourcing options
- Pexels: search "AI technology", "data flow", "abstract technology dark"
- Pixabay: search "technology background", "digital data"
- Coverr: coverr.co (free)

## Fallback
If video is missing, the hero falls back to the dark gradient background.
The `poster` attribute points to `/images/hero-poster.jpg` — add a dark cinematic
still from the video (or any suitable dark tech image) as fallback for slow connections.
